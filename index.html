Quick Installation Guide

1

Install the package

Use npm to install the scrubbe package and its dependencies:

npm install scrubbe-dash --save
2

Initialize the dashboard

Create a new dashboard configuration file:

npx scrubbe-dash init

This will create a scrubbe.config.js file in your project root.

3

Configure your integrations

Edit the config file to add your integration credentials:

// scrubbe.config.js
4

Start the dashboard

Run the dashboard with your configuration:

npx scrubbe-dash start

Your dashboard will be available at http://localhost:3000
