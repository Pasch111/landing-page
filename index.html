Quick Installation Guide

Install scrubbe dasboard in minutes

1

Install the package

Use npm to install the dashboard package and its dependencies:

npm install scrubbe --save
2

Initialize the dashboard

Create a new dashboard configuration file:

npx scrubbe-dash init

This will create a scrubbe.config.js file in your project root.

3

Configure your integrations

Edit the config file to add your integration credentials:

// scrubbe.config.js module.exports = { port: 3000, integrations: { aws: { region: 'us-east-1', credentials: { accessKeyId: process.env.AWS_ACCESS_KEY, secretAccessKey: process.env.AWS_SECRET_KEY, }, }, azure: { connectionString: process.env.AZURE_CONNECTION_STRING, }, datadog: { apiKey: process.env.DATADOG_API_KEY, appKey: process.env.DATADOG_APP_KEY, }, }, };
4

Start the dashboard

Run the dashboard with your configuration:

npx scrubbe-dash start

Your dashboard will be available at http://localhost:3000
